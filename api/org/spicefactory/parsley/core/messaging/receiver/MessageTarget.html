Interfacepublic interface MessageTarget extends MessageReceiver
Implementors DefaultMessageBinding, DefaultMessageHandler, MappedCommandProxy

Represent a regular target for a message. This interface should be implemented by all general-purpose message receivers which are not interceptors or error handlers. Builtin implementations are MessageHandler, MessagePropertyHandler, MessageBinding and CommandProxy.
